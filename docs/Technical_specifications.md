# 기술 스펙

1. SSR(+SSG) vs CSR
    1. 사용자들의 유입이 활성화 되어야 하는 초기 단계의 플랫폼으로, 검색엔진에 노출되면 노출될 수록 좋은 사이트이다.
    2. CSR로 선택할 때는 랜딩 페이지, 혹은 앱에서 이벤트성 페이지나 마케팅성으로 뿌릴 때 쓰는 프로젝트의 경우 CSR로 선택해도 문제 없으나, 플랫폼이나 외부에 적극적인 노출을 고려해야 할 때에는 SSR을 선택하는게 합당하다.

✅ SSR
